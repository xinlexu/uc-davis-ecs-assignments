# ECS130 Coursework

Numerical linear algebra homework source extracted from the local ECS130 submission archives.

- `hw1`: triangular solve and LU decomposition helpers.
- `hw2`: curve fitting, QR helpers, and truss simulation code.
- `hw3`: eigenvalue, SVD, low-rank approximation, and vibration mode code with small required data files.

Generated PDFs, reports, and bytecode caches are intentionally omitted.
