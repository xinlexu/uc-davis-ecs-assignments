# ECS36C Coursework

Programming assignments and practice exercises for UC Davis ECS36C.

## Contents

- `Program1` through `Program4`: data structures and algorithm programming assignments.
- `PracticeExercises`: smaller C/C++ exercises plus a Connect-N project collected from the local UC Davis coursework folder.

Generated binaries, debugger bundles, IDE metadata, and macOS metadata files are intentionally ignored.
